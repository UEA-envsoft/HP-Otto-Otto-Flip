<h2>HP Otto: Otto Flip</h2>
<img width="219" height="200" alt="Otto Flip" src="https://github.com/user-attachments/assets/e44bf3c2-6566-45c8-b31f-2c56810fdbd3" />

This modification gives the HP Otto robot a cam activated, elastic band powered, ball flipper 

STL files and build instructions can be found on [printables.com](https://www.printables.com/model/1516836-hp-otto-otto-flip)

rc.py is a modified version of rc.py enabling control of the flipper via the new HP Otto phone app <i>(currently (Mar 2026) in beta test and only available to purchasers of the HP Otto Starter https://www.facebook.com/groups/ottodiy/posts/3923882564421879/)</i>  

<img width="512" height="229" alt="otto flip phone control" src="https://github.com/user-attachments/assets/3c73488e-d870-4faa-bc73-7f7a6f7a5234" />

 The joystick control controls movement in any given direction, the D-pad control controls the flipper.

<h3>Instructions</h3>  
Just drop rc.py in Otto's root directory.
